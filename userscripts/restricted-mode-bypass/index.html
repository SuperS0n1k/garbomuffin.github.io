for YouTube
This is certainly against the YouTube TOS. Use at your own risk.

Get around restricted mode on YouTube!

Installation

  1. Install TamperMonkey (chrome) or GreaseMonkey (firefox)
  2. Click the button below this text.
  3. In the new popup click Install.
  4. YOU'RE DONE.
  5. It will update iteslf.

Usage

  • If a video is restricted it will create a button titled "Watch it anyway"
  • Click it to watch it anyway!
  • It will now start playing the video and load metadata.
  • Be aware that initially it might take a while to load, sometimes >10 seconds. This is normal! Be patient.

After reading the above text, click here to install!
