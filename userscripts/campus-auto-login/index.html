Campus Auto Login (v3.0)

Auto login to the District 112 campus portals!

Installation

  1. Install TamperMonkey.
  2. Click the giant link below and then "Install".
  3. You're done. It should (in theory?) update automatically.

Click here to install it.

Usage

  1. Enter your username and password into the boxes like usual.
  2. Instead of clicking "Submit", click the "Remember Me" button this has created.
  3. You're good to go, from now on it will automatically set the fields and click submit.

It will detect incorrect passwords or captchas and stop submitting and reset passwords. (you will have to tell this to remember them again with the "Remember Me" button)

It supports both the old and the new portals with support for empower planned. (but not in yet)

Changelog

v3.0:

  • Complete rewrite in typescript instead of javascript. (transpiles to javascript)
  • The method of saving passwords has changed. You now enter your passwords into the boxes then click "Remember Me"
