Exact YT Likes

See how many likes a YouTube video REALLY has.

The geniuses over at YouTube thought it would be a great idea if they rounded likes to the nearest thousand or whatever.

This has some problems, like in here where people are no longer able to balance the likes and dislikes!

The fix? Install this really simple user script! Install TamperMonkey (chrome) or GreaseMonkey (firefox) from their respective stores to install it.

Click here to install it.

(If you have tampermonkey/greasemonkey installed correctly all you have to do is click)

It's not a virus. If you don't believe me the script itself is only 20 lines of simple javascript.
