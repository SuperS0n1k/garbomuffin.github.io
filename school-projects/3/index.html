Instructions

Enter the image code below and click the button. The color pallete may need to be adjusted using some more buttons below.

If you hover over a color in the result you can see the coordinates. If you hover over an axis you can see what it covers.

Features

  • Any grid size will work (if it fits in your screen)
  • Custom palettes
  • Somewhat lenient syntax parsing as long as you use the basic 'AMOUNT (COLOR), ' format
  • If it detects an error in the code it will tell you where it was
  • A bunch of stuff to make it easy to find coordinates
