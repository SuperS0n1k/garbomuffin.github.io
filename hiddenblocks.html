Select a project file and all the blocks will become hidden. In theory it shouldn't break the project.

Use at your own risk. Note that THIS PROCESS CANNOT BE EASILY REVERSED, AND IS EFFECTIVELY PERMANENT. THE EDITOR ALSO BECOMES NEARLY IMPOSSIBLE TO USE. MAKE BACKUPS OF YOUR PROJECT BEFORE USING.

How it works: it sets the "shadow" property of all blocks in the project to true. Apparently that makes them invisible for some reason.
