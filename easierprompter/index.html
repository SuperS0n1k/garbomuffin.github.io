What is this?

This is a simple teleprompter. You enter your script above then press the start button above.

How is this better than any other prompter out there?

  • No sign in.
  • Written in pure javascript with no additional plugins.
  • All client side. I never see your scripts.
  • No ads.

Are there keyboard shortcuts?

Yes. In the prompter the following shortcuts exist:

  • Space - start/stop
  • Escape - stop and move prompter to top OR exit if already at the top
  • Up - increase speed
  • Down - decrease speed
