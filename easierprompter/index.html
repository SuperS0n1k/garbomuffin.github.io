What is this?

This is a simple teleprompter. You enter your script above then press the start button above. No ads and no login. It just works.

How is this better than any other prompter out there?

  • Free.
  • No login.
  • No ads.
  • All client side. Your scripts are always private and I never get to see them.
  • Written in pure javascript with no additional plugins. (eg. Flash)
  • Works in versions of Internet Explorer 10 and later. (and any version of Chrome/Firefox released in the past few years)
  • Work offline in supported browsers as long as you loaded the page once. (browsers that are green on here)
  • Open source

Keyboard shortcuts

  • Space - Start/Stop
  • Escape - Stop and move prompter to top OR exit if already at the top
  • Up Arrow - Increase speed
  • Down Arrow - Decrease speed
  • Hold R - Reverse while R is held
