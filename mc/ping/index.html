/ping

This is a small (just 8KB) and configurable plugin that just adds /ping [player] to a Bukkit server.
It should work on any version of Minecraft (tested in 1.13, 1.13.2, 1.10.2, 1.12.2) and any fork of Bukkit. (tested in Paper)

Click To Download

Features

  • Get your ping with /ping
  • Get the ping of another player with /ping <player name>
  • Permission support
  • Everything is configurable

Config

The config file (config.yml in the Ping folder) is heavily commented and should be easy to understand.

Permissions

There is one permission: ping.command. It allows access to /ping. It is given to everyone by default.
