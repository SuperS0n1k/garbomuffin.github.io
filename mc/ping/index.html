Ping (v0.9, supports any MC)

This is a simple and configurable plugin that simply adds /ping [player] to a Bukkit server.

Click To Download

(Any Minecraft version, works in 1.13)

Features

  • Get your ping with /ping
  • Get the ping of another player with /ping <player>
  • Permission support
  • Config support for just about everything
  • Will show a warning when getting the ping of a player joined very recently because it can be very inaccurate. (configurable and disableable)
  • Works with just about any version of Bukkit/Spigot/Paper (tested in 1.13, 1.12.2, and 1.10.2 but should work in any version)

Config

Download the plugin and edit the config.yml in the Ping folder to view the config. It is commented and should be easy to understand.

Any messages can be configured with color code support.

Permissions

There is one permission - ping.command. It allows access to /ping. It is given to everyone by default.
